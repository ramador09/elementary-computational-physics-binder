The notebooks here are published **without worked solutions** — each exercise
keeps its problem statement and validation checks, but the reference-solution
code is removed (that part is your exercise) — and with stored outputs cleared:
run the cells to bring each notebook to life. The rendered book shows every
figure and result. If you would like the reference solutions — to teach from or
to check your own work — get in touch: <hello@ramador.me>.
